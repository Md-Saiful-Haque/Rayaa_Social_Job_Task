# RAAYA.Social - Short-Form Video Content Platform


## 📋 Project Overview

RAAYA.Social is a cutting-edge, responsive landing page for an innovative social media content company with a focus on producing short-form video content. The platform assists companies and individual brands in producing captivating video content for a variety of social media platforms by offering customized social media content packages that start at $99/month.

### 🎯 Key Features

A fully responsive layout that functions flawlessly on desktop, tablet, and mobile devices is known as responsive design. The Hamburger menu offers the best mobile browsing experience.Expandable and collapsible FAQ pieces with seamless transitions make up the interactive FAQ section. A thorough showcase of innovative services and bundles is called the "Service Showcase." There are specific categories for both personal and company brand videos. A display of the social media platforms that are supported, such as YouTube, Instagram, and TikTok. An integrated contact form for inquiries is located in the footer.

### Technologies Used

- HTML5
- CSS3 
- JavaScript (vanilla)
- Google Fonts (Inter, Nunito, Poppins, Raleway)
- Font Awesome Icons
